从感知到神经网络

  • 感知机需要人工设置合适的权重，神经网络可以解决这个问题
  • 神经网络重要性质：它可以自动地从数据中学习到合适的权重参数

神经网络图例

  • 中间层有时候称为隐藏层

激活函数

  • 对输入信号的总和进行转换输出的函数，即激活函数是连接感知机和神经网络的桥梁

线性函数和非线性函数

  • 线性函数：图像为一条笔直的直线
  • 非线性函数：图像不是笔直的直线
  • 因为线性函数总存在与之等效的”无隐藏层的神经网络“，例如线性函数h(x)=cx，则y(x)=h(h(h(x)))对应三层神经网络，可以用y(x)=ax这个没有隐藏的神经网络进行一次运算即可，其中a为c的三次方。为了发挥叠加层的优势，激活函数必须使用非线性函数

sigmoid函数和阶跃函数比较

不同点
  • 阶跃函数只能返回0或1，sigmoid函数可以返回0.731….等实数，与前面图像的中平滑性相关
  • 感知机中神经元之间流动的是二元信号，神经网络中流动的是连续性的实数值信号
共同点
  • 虽然二者平滑性有差异，但二者结果均为”输入小时，输出接近0，反之则接近1“
  • 二者的输出信息都在0到1之间
  • 二者均为非线性函数

输出层设计

    • 分类问题是数据属于哪一个类别的问题：区分图像中的男性或女性等

恒等函数

  • 恒等函数会将输入信号原样输出，和前面介绍的隐藏层激活函数一样，恒等函数进行的转换处理可以用一根箭头来表示

softmax函数

基础概念及实现

  • 分子是输入信号的指数函数，父母是所有输入信号的指数函数的和

  • 从图中可看出，输出层的各个神经元都受到了所有输入信号的影响

  • 函数实现：

    • def softmax(a):
    exp_a=np.exp(a)
    sum_exp_a=sum(exp_a)
    y=exp_a/sum_exp_a
    return y

函数特征

  • softmax函数的输出是0.0到1.0之间的实数，且输出值总和为1，即softmax函数的输出可以解释称“概率”
