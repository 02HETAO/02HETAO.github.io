  • 神经网络重要性质：它可以自动地从数据中学习到合适的权重参数

2.激活函数

  • 对输入信号的总和进行转换输出的函数，即激活函数是连接感知机和神经网络的桥梁

线性函数和非线性函数

  • 线性函数：图像为一条笔直的直线
  • 非线性函数：图像不是笔直的直线
  • 因为线性函数总存在与之等效的”无隐藏层的神经网络“，例如线性函数h(x)=cx，则y(x)=h(h(h(x)))对应三层神经网络，可以用y(x)=ax这个没有隐藏的神经网络进行一次运算即可，其中a为c的三次方。为了发挥叠加层的优势，激活函数必须使用非线性函数

sigmoid函数和阶跃函数比较

不同点
  • 阶跃函数只能返回0或1，sigmoid函数可以返回0.731….等实数，与前面图像的中平滑性相关
  • 感知机中神经元之间流动的是二元信号，神经网络中流动的是连续性的实数值信号
共同点
  • 虽然二者平滑性有差异，但二者结果均为”输入小时，输出接近0，反之则接近1“
  • 二者的输出信息都在0到1之间
  • 二者均为非线性函数

5.输出层设计

  • 机器学习的问题大致可以分为分类问题和回归问题
    • 分类问题是数据属于哪一个类别的问题：区分图像中的男性或女性等
    • 回归问题是根据某个输入预测一个（连续的）数组的问题：预测图像中这个人的体重
  • 注意：输出层所用的激活函数要根据求解问题的性质决定
    • 回归问题：恒等函数
    • 二元问题：sigmoid函数
    • 多元分类问题：softmax函数

恒等函数

  • 恒等函数会将输入信号原样输出，和前面介绍的隐藏层激活函数一样，恒等函数进行的转换处理可以用一根箭头来表示

softmax函数

基础概念及实现

  • 分子是输入信号的指数函数，父母是所有输入信号的指数函数的和

  • 从图中可看出，输出层的各个神经元都受到了所有输入信号的影响

  • 函数实现：

    • def softmax(a):
    exp_a=np.exp(a)
    sum_exp_a=sum(exp_a)
    y=exp_a/sum_exp_a
    return y

函数特征

  • softmax函数的输出是0.0到1.0之间的实数，且输出值总和为1，即softmax函数的，出可以解释称“概率”
  • 根据公式可知，神经网络只把输出值最大的神经元所对应的类别作为识别结果，即便使用softmax输出值最大神经元位置也不会变
  • 实际问题中，由于指数运算需要一定计算机运算量，因此输出层softmax函数一般会省略

输出层的神经元数量

  • 输出层的神经元数量需要根据待解决的问题来决定：
    • 分类问题：神经元数量一般设定为类别数量，如一个输入图像，预测是图中的数字0-9的哪一个问题，即10类别

6.手写数字识别

  • 神经网络的“推理处理”，也称神经网络的前向传播
  • 求解机器学习问题的步骤：学习和推理两个阶段
  • 神经网络解决问题步骤：首先使用训练数据进行权重参数的学习；进行推理时使用刚才学到的参数，对输入数据进行分类

MNIST数据集

  • 这里使用的数据集是MNIST手写数据集，MNIST是机器学习最有名的数据集之一，应用于实验到论文研究各个场合
  • 一般使用方法：先训练图像进行学习，再用学习到的模型度量能在多大程度上对测试图像进行正确分类

特征

  • MNIST的图像数据是28像素*28像素
  • 数据集由0-9的数字图像构成，训练图像有6万张，测试图片有1万张
  • load_mnist函数以“(训练图像 ,训练标签 )，(测试图像，测试标签 )”的形式返回读入的MNIST数据

本章小结

  • 神经网络中的激活函数使用平滑变化的sigmoid函数或ReLU函数
  • 通过巧妙地使用NumPy多维数组，可以高效地实现神经网络
  • 机器学习的问题大体上可以分为回归问题和分类问题
  • 关于输出层的激活函数，回归问题中一般用恒等函数，分类问题中一般用softmax函数
  • 分类问题中，输出层的神经元的数量设置为要分类的类别数
  • 输入数据的集合称为批。通过以批为单位进行推理处理，能够实现高速的运算
